Web-Based Logical Assistant

An interactive reasoning & set operations solver built with C++ logic

Project Overview

This project is a C++ backend-powered web assistant designed to help students solve logical reasoning and set operation problems interactively. The assistant provides step-by-step outputs based on user input, offering a quick and educational tool for logic-based tasks.

Technologies Used

  • C++ (Core Logic)
  • HTML & CSS (Frontend UI)
  • JavaScript (Basic interactivity)
