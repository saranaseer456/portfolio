Arduino Rocket Guidance System

Microcontroller-based directional control using real-time stabilization

Project Overview

This project is a prototype of a rocket guidance system using an Arduino microcontroller. It uses sensors (like MPU-6050) and servo motors to correct the rocket's direction in real time. The purpose is to simulate stabilization logic for small UAVs or model rockets. This is a fully functional demo to show how automatic directional correction works in flight.

Technologies Used

  • Arduino Uno
  • MPU-6050 (Gyro + Accelerometer)
  • Servo Motors
  • C++ using Arduino IDE
