Terminal-Based Space Invaders

A C++ game with object-oriented programming, enemy logic & terminal UI

Project Overview

This is a fully object-oriented, terminal-based Space Invaders game coded in C++. It includes features like player movement, bullet shooting, enemy generation, level progression, and win/loss conditions. The entire logic is designed using OOP principles such as inheritance and encapsulation to demonstrate game architecture in a clean and maintainable way.

Technologies Used

  • C++ (Object-Oriented Programming)
  • Console Input/Output (Platform Independent)
  • Data Structures for Game Logic
